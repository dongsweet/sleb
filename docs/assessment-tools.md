# 评估工具说明

> SLEB 提供四种能效评估工具，复杂度从低到高

---

## 1. THEEA（Total Home Energy Assessment）

**复杂度：** ⭐ 最简单
**目标用户：** 住宅业主
**入口：** `/tools/theea`

### 说明

住宅能效评估工具，评估整栋住宅的能源使用效率。

### 输入参数

- 住宅类型（组屋/排屋/公寓/别墅）
- 建筑面积
- 住户人数
- 电器清单及功率
- 空调使用时长
- 照明系统类型

### 输出

- 综合能效评分
- 各系统能耗分解
- 改进建议

### 备注

- 参数固定，算法简单
- 适合个人用户快速评估

---

## 2. SMEEA（SME Energy Assessment）

**复杂度：** ⭐⭐⭐ 中等
**目标用户：** 中小企业
**入口：** `/tools/smeea`

### 说明

中小企业能效评估工具，适用于办公室、小型工厂等商业场景。

### 输入参数

- 企业类型
- 建筑面积和楼层
- 营业时间
- HVAC 系统类型
- 照明系统
- 生产设备清单
- 月度能耗数据

### 输出

- 能效评分（对比同类企业）
- 能耗分项报表
- 节能改进建议
- **Standard/Premium 会员可下载报告**

### 备注

- 需要更多行业基准数据
- 报告生成是关键功能

---

## 3. BEEA（Building Energy Efficiency Assessment）

**复杂度：** ⭐⭐⭐⭐⭐ 最复杂
**目标用户：** 大型商业建筑
**入口：** `/tools/beea`

### 说明

建筑能效评估工具，适用于大型商业楼宇的全面能效评估。

### 输入参数

- 建筑基本信息（类型、面积、竣工年份、Green Mark 等级）
- **按系统分类的能耗数据**：
  - ACMV（空调通风）
  - 照明
  - 电梯
  - 其他系统
- 月度/年度能耗数据
- 建筑运行时间表
-  Occupancy 数据

### 输出

- 综合 EUI（Energy Use Intensity）评分
- 按系统分类的能耗饼图
- 按月分布的柱状图
- 能效等级对比
- 详细的节能改进方案
- 投资回报分析

### 备注

- 参数最多，算法最复杂
- 需要大量的基准数据和行业参数
- 原站算法写死在代码中，新系统需要可配置

---

## 4. AI Calculator

**复杂度：** ⭐⭐⭐⭐ 高级
**目标用户：** 所有会员
**入口：** `/tools/ai-calculator`

### 说明

AI 驱动的 ROI 计算器和节能方案推荐工具。

### 输入参数

- 建筑基本信息
- 当前能耗数据
- 预算范围
- 期望的投资回收期
- 偏好技术类型

### 输出

- ROI 计算结果
- 推荐的技术/产品方案
- 预期节能效果
- 投资回收期分析
- 政府补贴匹配

### 备注

- **核心 AI 功能** — 需要接入大模型或知识库
- 推荐结果基于平台上的技术/产品目录
- 需要与 AI 知识库联动

---

## 5. 算法配置（后台）

### 需求

从会议纪要确认：**需要在后台可以更换算法/调整参数**

### 实现方案

1. **参数表**：在 `system_parameters` 或 `assessment_parameters` 表中存储各工具的参数
2. **算法版本**：每个工具可以有多个算法版本，后台选择生效版本
3. **参数结构**：
   ```json
   {
     "tool": "beea",
     "version": "2.0",
     "parameters": {
       "base_eui_threshold": 150,
       "acmv_weight": 0.45,
       "lighting_weight": 0.20,
       "elevator_weight": 0.10,
       "other_weight": 0.25,
       "benchmark_sources": ["BCA_2024", "SLEB_2024"]
     }
   }
   ```

### 后台管理页面

- 查看所有工具的当前参数
- 编辑参数并保存新版本
- 查看参数变更历史
- 回滚到之前版本

## 6. 数据源

| 数据类型 | 来源 | 更新频率 |
|----------|------|----------|
| 建筑基础数据 | directoryEntries 表 | 手动 |
| 月度能耗数据 | Excel 导入 / API 接口 | 月度 |
| 水电数据 | 新加坡 SP（公用事业）| 每日 |
| 行业基准 | BCA / SLEB 发布 | 年度 |
| 技术/产品目录 | 会员提交 | 实时 |
| 政府补贴 | 政策文件 | 手动更新 |

## 7. Smart Advisor（AI 推荐）

### 原站实现

写死的建议列表，修改困难。

### 新系统实现

**AI 知识库驱动的个性化推荐：**

1. 根据建筑数据（类型、面积、EUI、Green Mark 等级等）生成个性化建议
2. AI 知识库包含：
   - 平台上的所有技术/产品信息（从会员上传的 brochure 自动提取）
   - 政府补贴政策（prompt 中配置）
   - 行业最佳实践
3. 管理员可以更新 AI 的"员工手册"（prompt）来调整推荐策略
4. 新产品上线时，AI 自动学习并纳入推荐

### 技术架构

```
建筑数据 → AI API → 知识库检索 → Prompt 组装 → 生成建议
                              ↑
                    管理员配置的 prompt
```
