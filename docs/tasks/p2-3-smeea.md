# P2-3：SMEEA 企业综合评估

| 字段 | 值 |
|------|------|
| 编号 | P2-3 |
| 优先级 | P2 — 评估工具 |
| 预估工时 | 6h |

## 背景

SMEEA（Sustainability Management Evaluation for Energy Awareness）面向企业/企业园区。评估范围包括建筑能耗 + 碳排 + 可持续管理实践。

## 目标

实现企业级综合可持续性评估。

## 涉及文件

1. **新建** `apps/web/app/smeea/page.tsx` — 评估入口页
2. **新建** `apps/web/app/components/assessments/SmeeaForm.tsx` — 多步骤表单
3. **新建** `apps/api/src/modules/assessments/smeea.ts` — 计算逻辑
4. **已有** `packages/db/src/schema.ts` — `assessmentRecords` 表（type: smeea）

## 表单步骤

| 步骤 | 标题 | 内容 |
|------|------|------|
| 1 | 企业信息 | 公司名称、行业、员工数、运营建筑 |
| 2 | 能源管理 | 是否使用 EMS、是否做能源审计、有无人负责 |
| 3 | 碳排管理 | 是否计算碳足迹、是否有减排目标 |
| 4 | 可持续实践 | 节水措施、废弃物管理、绿色采购 |
| 5 | 认证与合规 | Green Mark、ISO 14001、ISO 50001 |
| 6 | 结果 | 综合评分、碳排数据、改进建议 |

## 评分维度

- 能源管理：30%
- 碳排管理：30%
- 可持续实践：20%
- 认证与合规：20%

## 实现要点

- 需要 Premium 会员权限
- 大量是选择题（是/否/部分），较少数字输入
- 结果包含雷达图（四维度评分）
- 支持关联多个建筑的数据
