# P4-2：智能建议引擎

| 字段 | 值 |
|------|------|
| 编号 | P4-2 |
| 优先级 | P4 — AI 功能 |
| 预估工时 | 5h |

## 背景

Smart Advisor 基于用户建筑的实际能耗数据，生成个性化的节能建议。是 My Building 仪表板中的一个 Tab。

## 目标

1. 分析建筑能耗数据
2. 生成改进建议（包含预期节省）
3. 展示在 My Building 仪表板的 Smart Advisor Tab

## 涉及文件

1. **新建** `apps/web/app/components/tabs/SmartAdvisorTab.tsx` — Smart Advisor 组件
2. **新建** `apps/api/src/modules/advisor/routes.ts` — 建议 API
3. **新建** `apps/api/src/modules/advisor/engine.ts` — 建议引擎

## 建议类型

| 类型 | 示例 | 预期节省 |
|------|------|---------|
| HVAC 优化 | 升级变频空调 | 20-30% |
| 照明改造 | LED 替换 + 智能控制 | 15-25% |
| 保温改造 | 外墙保温层 | 10-15% |
| 能源管理 | EMS 系统安装 | 5-10% |
| 可再生能源 | 太阳能板 | 30-50% |

## 引擎逻辑

```
输入：建筑类型 + 面积 + 各系统能耗数据
  ↓
规则引擎（基于行业基准和最佳实践）
  ↓
AI 模型（可选：基于 LLM 的自然语言建议）
  ↓
输出：排序后的建议列表（按 ROI 排序）
```

## API 端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/advisor/:buildingId` | GET | 获取建议 |

## 实现要点

- 第一阶段：规则引擎（if-then 逻辑）
- 第二阶段：接入 LLM 生成自然语言建议
- 建议按 ROI（投资回报率）排序
- 每个建议包含：标题、描述、预期节省（%和金额）、投资估算

## 验收标准

- [ ] 能根据建筑数据生成建议
- [ ] 建议按 ROI 排序
- [ ] 建议内容合理
- [ ] 图表展示节省潜力
