# P5-2：响应式设计

| 字段 | 值 |
|------|------|
| 编号 | P5-2 |
| 优先级 | P5 — 设计增强 |
| 预估工时 | 3h |

## 背景

所有页面需要在移动端和平板上正常显示。

## 目标

1. 移动端导航（汉堡菜单）
2. 所有页面的响应式布局
3. 表单在移动端的良好体验

## 涉及文件

1. **修改** `apps/web/app/components/SiteChromeShell.tsx` — 响应式导航
2. **修改** `apps/web/app/styles.css` — 全局响应式样式

## 断点设计

| 断点 | 宽度 | 设备 |
|------|------|------|
| mobile | < 640px | 手机 |
| tablet | 640-1024px | 平板 |
| desktop | > 1024px | 桌面 |

## 实现要点

- 汉堡菜单：移动端隐藏导航链接，点击展开
- 卡片网格：桌面 4 列 → 平板 2 列 → 手机 1 列
- 表单：全宽输入
- 地图：移动端全屏
- 表格：横向滚动或卡片模式

## 验收标准

- [ ] 移动端导航正常
- [ ] 所有页面响应式
- [ ] 表单移动端友好
- [ ] 地图移动端适配
- [ ] 表格移动端适配
