# P1-4：新闻页面

| 字段 | 值 |
|------|------|
| 编号 | P1-4 |
| 优先级 | P1 — 公共页面 |
| 预估工时 | 3h |

## 背景

原站有大量新闻文章，需要实现列表页和详情页。当前 `/news` 路由已存在但使用 mock 数据。

## 目标

1. 新闻列表页：卡片列表 + 分页
2. 新闻详情页：文章正文 + 图片

## 涉及文件

1. **已有** `apps/web/app/news/page.tsx` — 需改造为真实数据
2. **新建** `apps/web/app/news/[slug]/page.tsx` — 新闻详情
3. **新建** `apps/api/src/modules/content/routes.ts` — 内容 API（已存在，需完善）
4. **已有** `packages/db/src/schema.ts` — `contentItems` 表（type: news）

## API 端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/content` | GET | 内容列表（type, tag 筛选 + 分页）|
| `/api/content/:id` | GET | 内容详情 |

## 实现要点

- 使用已有的 `[...slug]` 动态路由或新建独立路由
- 新闻详情页面需要富文本渲染（Markdown → HTML）
- 图片存储在 MinIO
- 按发布日期倒序排列

- [ ] 标签筛选功能正常
