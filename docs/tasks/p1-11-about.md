# P1-11：关于页面

| 字段 | 值 |
|------|------|

| 优先级 | P1 — 公共页面 |
| 预估工时 | 1h |

## 背景

About 页面展示 SLEB 的使命、愿景、团队等信息。这是纯静态内容。

## 目标

实现 `/about` 页面。

## 涉及文件

1. **新建** `apps/web/app/about/page.tsx` — 关于页面
2. **新建** `packages/shared/src/content.ts` — 关于内容（静态）

## 页面内容

- 使命陈述
- SLEB 简介
- 资助方信息（NRF、BCA）
- 团队成员/合作机构
- 时间线（发展历程）

## 实现要点

- 纯静态内容，无需 API
- 可以从原站截图/复制文本
- 使用 Hero + 内容区块的布局

## 验收标准

- [ ] 页面内容完整
- [ ] 排版美观
- [ ] 移动端响应式
