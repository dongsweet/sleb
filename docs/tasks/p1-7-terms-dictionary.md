# P1-7：术语词典页面

| 字段 | 值 |
|------|------|
| 编号 | P1-7 |
| 优先级 | P1 — 公共页面 |
| 预估工时 | 2h |

## 背景

可持续建筑领域的专业术语词典，帮助用户理解行业术语。

## 目标

1. 术语列表页（`/terms-dictionary`）— A-Z 索引 + 搜索
2. 术语详情页（`/terms-dictionary/[slug]`）

## 涉及文件

1. **新建** `apps/web/app/terms-dictionary/page.tsx` — 术语列表
2. **新建** `apps/web/app/terms-dictionary/[slug]/page.tsx` — 术语详情
3. **新建** `packages/shared/src/content.ts` — 术语数据（静态）
4. **已有** `packages/db/src/schema.ts` — `contentItems` 表（type: glossary）

## 页面布局

```
┌─────────────────────────────────────────────────┐
│  Terms Dictionary                               │
│  [搜索框]                                       │
│                                                 │
│  A  B  C  D  E  F  G  H  I  J  K  L  M  ...   │
│                                                 │
│  ┌─────────────────────────────────────────┐    │
│  │ A | 42 terms                            │    │
│  │ ─────────────────────────────────────── │    │
│  │ Ambient Cooling     [view]              │    │
│  │ Anaerobic Digestion [view]              │    │
│  │ ...                                     │    │
│  └─────────────────────────────────────────┘    │
└─────────────────────────────────────────────────┘
```

## 实现要点

- 术语数据可以放在 shared 包作为静态数据（不频繁更新）
- A-Z 索引：点击跳转到对应字母区域
- 搜索：实时过滤，输入时显示匹配结果
- 详情页：术语名称、定义、相关术语链接

## 验收标准

- [ ] A-Z 索引功能正常
- [ ] 搜索功能正常（实时过滤）
- [ ] 详情页展示术语定义
- [ ] 相关术语链接可用
